## 网络安全技能

### Web 安全

- 熟练掌握 OWASP Top 10 漏洞的检测与利用，包括 SQL注入、XSS、CSRF、SSRF、XXE、文件上传、文件包含、命令注入、反序列化、SSTI 等。
- 了解 Java 反序列化利用链（CommonsCollections、CommonsBeanutils、Spring AOP 等），熟悉 Shiro、Fastjson、WebLogic 等中间件的漏洞利用。
- 熟悉 JNDI 注入原理及高版本 JDK 的绕过技巧。
- 掌握 PHP 代码审计中常见危险函数与 POP 链构造思路。
- 熟悉内存马（Filter型、Servlet型、Listener型、Valve型）的原理与检测方法。

### 内网渗透

- 掌握内网信息收集、横向移动、权限维持的技术体系。
- 了解 Kerberos 协议攻击（黄金票据、白银票据、Kerberoasting、DCSync）。
- 熟悉代理转发技术（frp、Neo-reGeorg、Chisel）与隧道搭建。

### 免杀与对抗

- 了解杀软/EDR 检测原理及常见的静态免杀、动态免杀、行为免杀技术。
- 熟悉 Cobalt Strike 的 malleable C2 profile 定制与流量隐匿。
- 了解 Office 宏攻击、白加黑 DLL 劫持、银狐木马等 APT 攻击手法。

### 流量分析

- 能够通过 Wireshark、TShark 对 HTTP/DNS/TLS 流量进行分析。
- 能够识别常见 WebShell 管理工具（蚁剑、冰蝎、哥斯拉）的通信流量特征。

### CTF 竞赛

- 多次参加各类 CTF 竞赛（Web/Misc/Crypto 方向），具有漏洞复现和 EXP 脚本编写能力。
- 熟悉 CTF 中 PHP 伪协议、无参 RCE、无列名注入等进阶利用技巧。
