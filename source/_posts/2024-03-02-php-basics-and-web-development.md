# PHP基础与Web开发详解

本文详细介绍了PHP在Web开发中的基础知识，包括静态与动态网站的区别、Apache服务器配置、PHP核心概念等内容。

<!-- more -->

## 静态网站和动态网站

### 静态网站
- **定义**：保存在服务器上的一个文件
- **特点**：内容固定，不会根据用户操作实时变化

### 动态网站
- **定义**：与数据库联系，根据用户操作实时更新
- **特点**：内容可变，常见php、asp等结尾的文件

## 页面解析过程

### 静态页面解析流程

1. 浏览器根据URL用DNS服务器解析成IP地址
2. 根据IP地址访问服务器
3. 用端口号找到Apache进程
4. Apache根据URI（URL中的目标机器的文件夹的路径）找到网页文件夹
5. 进行解析，最后返回给浏览器
6. 浏览器解析给客户端

### 动态页面解析流程

相比静态页面，动态页面多了一个重要步骤：
- 在Apache web服务器中交给PHP引擎解析（可能会交给数据库）

**注意**：Apache只能解析HTML的内容，所以需要有个PHP解析引擎。

## 常见文件扩展名和目录结构

### 文件扩展名
- **bin**：binary二进制缩写，Windows下的可执行文件
- **conf**：配置文件目录（configuration）
- **lib**：核心文件夹（Library）静态链接库
- **dll**：Dynamic Link Library（动态链接库）文件

### Apache目录结构
- **htdocs**：Apache默认的主机地址（网络根目录）
- **modules**：模块，可以加载到conf里面
- **conf**：配置文件目录

### 服务状态指示
- **绿色**：表示开启成功
- **黄色**：表示端口被占用
- **ab.exe**：压力测试工具，测试能够承载多少用户

### 库文件类型
- **lib（Library）**：静态链接库
- **dll文件**：动态链接库文件，包含了程序所需的代码和数据。与静态链接库不同，动态链接库可以在程序运行时动态加载，使得程序的内存占用更小，同时也方便了程序的更新和维护。

## Apache配置

### 基本配置
- **httpd**：服务器服务名
- **配置文件**：httpd.conf文件

### 常用命令
```bash
# 查询模块的使用
httpd.exe -M
# static: 静态加载
# shared: 只有使用的时候才会加载

# 检查配置文件是否能够使用
httpd.exe -t
# syntax ok: 证明没有语法错误
```

### 配置修改
- **网站名字**：用编辑器打开conf配置文件进行更改httpd.conf文件
- **端口配置**：可以单独实现
  ```apache
  Servername localhost:80
  listen 80  # 可以进行更改
  ```

## PHP核心概念

### 1. htmlspecialchars()方法

**定义**：在HTML中，由于小于号和大于号被用作标签的开始和结束标记，因此不能直接在HTML代码中使用它们。为了显示这些字符，我们需要使用它们的实体表示。

**名称方式**：以"&"开头，后面紧跟由英文字母组成的名称

**常见HTML实体**：
- `&lt;` - 小于号 (&lt;)
- `&gt;` - 大于号 (&gt;)
- `&amp;` - 和号 (&amp;)
- `&quot;` - 双引号 (&quot;)
- `&apos;` - 单引号 (&apos;)

### 2. PHP安全函数

`htmlspecialchars()`函数的作用：
- 将特殊字符转换为HTML实体
- 防止XSS攻击
- 确保用户输入的安全显示

**语法**：
```php
htmlspecialchars(string, flags, encoding, double_encode)
```

**参数说明**：
- **string**：必需，要转换的字符串
- **flags**：可选，规定如何处理引号、无效的编码以及使用哪种文档类型
- **encoding**：可选，规定要使用的字符集
- **double_encode**：可选，布尔值，规定是否编码已存在的HTML实体

### 3. PHP与Web安全

在Web开发中，安全性是一个重要考虑因素：

**输入验证**：
- 始终验证和清理用户输入
- 使用适当的过滤函数
- 防止SQL注入攻击

**输出编码**：
- 使用`htmlspecialchars()`等函数
- 防止XSS攻击
- 确保数据安全显示
