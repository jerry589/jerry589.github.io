# Linux系统管理学习笔记

本文整理了Linux操作系统的核心概念和系统管理技能，涵盖了目录结构、常用命令、权限管理、用户管理、磁盘管理等重要内容。

## 操作系统概述

操作系统是计算机的重要组成部分，是一个系统软件，负责管理和控制计算机的硬件和软件资源。

### 主要操作系统类型

| 操作系统类型 | 描述 | 常见版本/产品 |
|:----------:|:--------------------------------:|:---------------------------------------------:|
| Windows | 由微软公司开发的个人电脑操作系统，提供直观的图形用户界面 | Windows 10, Windows 8.1, Windows 7 |
| macOS | 由苹果公司开发的Mac电脑专用操作系统，设计优雅，用户体验良好 | macOS Catalina, macOS Mojave, macOS High Sierra |
| Linux | 自由和开放源码的类Unix操作系统，稳定性和安全性高 | Ubuntu, CentOS, Debian, Fedora |
| UNIX | 功能强大、稳定可靠的操作系统，常用于服务器和高性能计算 | AIX, Solaris, FreeBSD, OpenBSD |
| Android（移动端） | 由谷歌公司开发的开放源代码移动操作系统，广泛应用于智能手机和平板电脑 | Android 11, Android 10, Android 9 Pie |
| iOS（移动端） | 由苹果公司开发的专有移动操作系统，只能在苹果公司的设备上运行 | iOS 14, iOS 13, iOS 12 |

## Linux系统目录结构

Linux采用树形目录结构，以根目录（/）为起点，各个目录有特定的用途：

| 目录 | 描述 |
|:-------------:|:---------------------------------------------------------------------:|
| **/boot** | 存放**启动Linux时**使用的内核文件、连接文件及镜像文件 |
| **/etc** | 存放系统所需的**配置文件**和子目录列表。更改这里的文件可能会导致系统无法启动 |
| **/lib** | 存放基本代码库，类似于Windows中的DLL文件 |
| **/sys** | 集成proc、devfs和devpts文件系统的信息，反映内核设备树 |
| **/bin** | 存放最常用的程序和指令，如ls等 |
| **/sbin** | 存放只有系统管理员能使用的程序和指令 |
| **/dev** | 存放Linux的外部设备 |
| **/media** | 存放识别后的其他设备，如U盘、光驱等 |
| **/mnt** | 临时挂载其他文件系统的目录 |
| **/run** | 临时文件系统，存储系统启动以来的信息 |
| **/lost+found** | 系统非法关机后存放一些文件的目录 |
| **/tmp** | 存放临时文件的目录 |
| **/root** | 系统管理员的用户主目录 |
| **/home (~)** | 用户的主目录，**以用户的账号命名** |
| **/usr** | 存放用户的应用程序和文件，类似于Windows的**Program Files**目录 |
| **/usr/bin** | 存放系统用户使用的应用程序与指令 |
| **/usr/sbin** | 存放超级用户使用的高级管理程序和系统守护程序 |
| **/usr/src** | 内核源代码的默认放置目录 |
| **/var** | 存放经常修改的数据，如程序运行的**日志文件** |
| **/proc** | 虚拟目录，管理内存空间，是系统内存的映射 |
| **/opt** | 存放额外安装的软件 |
| **/srv** | 存放服务启动后需要提取的数据（不用服务器则为空） |

### 系统启动过程

1. **BIOS开机自检**
2. 按照BIOS设置的启动设备启动
3. 操作系统接管硬件，读入 **/boot 目录下的内核文件**
4. 运行init进程（系统所有进程的起点）
5. **sync**将内存中的数据与磁盘同步

## Linux命令基础

### 命令格式

| 组成部分 | 描述 | 示例 |
|:------:|:------------------------------------------------:|:-----------:|
| **命令** | 要**执行的命令**或操作 | `ls` |
| **选项** | 指定命令的**具体行为**或修改其默认行为 | `-l` |
| **参数** | 提供命令操作**所需的内容或数据（在哪里执行）** | `/home` |
| **命令格式** | `command [-options] [parameter1] [parameter2] ...` | `ls -l /home` |

### 常用命令

| 命令 | 英文全拼 | 描述 |
|:-----:|:------------------:|:--------------------:|
| `ls` | list files | 列出目录内容，包括文件和子目录 |
| `cd` | change directory | 切换当前工作目录 |
| `pwd` | print work directory | 显示当前工作目录的完整路径 |
| `mkdir` | make directory | 创建一个新的目录 |
| `rmdir` | remove directory | 删除一个空的目录 |
| `cp` | copy file | 复制文件或目录到指定位置 |
| `rm` | remove | 删除文件或目录 |
| `mv` | move file | 移动文件或目录到指定位置，或重命名文件或目录 |

### 常用选项

| 选项 | 描述 |
|:----:|:----------------------:|
| `-a` | 列出所有文件，包括隐藏文件（以`.`开头的文件） |
| `-d` | 仅列出目录本身，而非目录内的文件内容 |
| `-l` | 长格式显示，包括文件属性、权限、大小等信息 |

## Linux文件权限管理

### 文件属性解读

Linux文件权限由10个字符组成：
- 第1个字符：文件类型（d表示目录，-表示普通文件）
- 第2-4个字符：所有者权限
- 第5-7个字符：所属组权限
- 第8-10个字符：其他用户权限

### 权限表示

- **r（read）**：读权限，数值为4
- **w（write）**：写权限，数值为2
- **x（execute）**：执行权限，数值为1
- **-**：无权限，数值为0

### 权限修改命令

**chown（change owner）**：修改所属用户与组
```bash
chown [-R] 所有者:属组名 文件名
chown bin install.log  # 将文件所有者改为bin用户
```

**chmod（change mode）**：修改用户权限
```bash
chmod [-R] xyz 文件或目录
# 例如：chmod 770 filename
# owner = rwx = 4+2+1 = 7
# group = rwx = 4+2+1 = 7  
# others = --- = 0+0+0 = 0
```

## 用户和用户组管理

### 用户管理命令

| 序号 | 命令 | 描述 | 主要选项 |
|:----:|:-------:|:-------:|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------:|
| 1 | useradd | 添加新的用户账号 | -c comment **注释**<br/>-d 目录 **指定用户主目录**<br/>-g 用户组 **指定用户组**<br/>-G 用户组 指定用户所属的**附加组**<br/>-s Shell文件 指定用户的登录Shell<br/>-u 用户号 指定用户的用户号 |
| 2 | userdel | 删除已有的用户账号 | -r (同时删除用户主目录) |
| 3 | usermod | 修改已有用户的信息 | -c -d -m -g -G -s -u -o -l 新用户名 |
| 4 | passwd | 指定和修改用户口令 | -l 锁定口令，即**禁用**账号<br/>-u 口令**解锁**<br/>-d 使账号无口令<br/>-f 强迫用户下次登录时**修改**口令 |

### 用户组管理

- 用户组的所有信息都存放在`/etc/group`文件中
- 可以通过相关命令管理用户组的添加、删除和修改

## 磁盘管理

Linux磁盘管理直接关系到整个系统的性能问题。

### 常用磁盘管理命令

**df（disk free）**：列出文件系统的整体磁盘使用量
```bash
df -h  # 以人类可读的格式显示
```

**du（disk used）**：检查磁盘空间使用量
```bash
du -sh /path/to/directory  # 显示目录总大小
```

**fdisk**：用于磁盘分区
```bash
fdisk -l  # 列出所有磁盘分区
```

### 文件系统操作

**磁盘格式化**：
```bash
mkfs [-t 文件系统格式] 装置文件名
```

**磁盘检验**：
```bash
fsck [-t 文件系统] [-ACay] 装置名称
```
