# Linux系统与Shell基础学习笔记

本文整理了Linux操作系统的基础知识，包括系统目录结构、常用命令、文件权限管理、用户管理等核心概念。

## 1. 操作系统概述

### 什么是操作系统
操作系统是计算机的重要组成部分，是一个系统软件，负责管理和控制计算机的硬件和软件资源。

### 主要操作系统类型

| 操作系统类型 | 描述 | 常见版本/产品 |
|------------|------|---------------|
| **Windows** | 微软开发的个人电脑操作系统，图形用户界面直观 | Windows 10, Windows 8.1, Windows 7 |
| **macOS** | 苹果开发的Mac专用操作系统，设计优雅 | macOS Catalina, macOS Mojave, macOS High Sierra |
| **Linux** | 自由开源的类Unix系统，稳定性和安全性高 | Ubuntu, CentOS, Debian, Fedora |
| **UNIX** | 功能强大、稳定可靠，常用于服务器 | AIX, Solaris, FreeBSD, OpenBSD |
| **Android** | 谷歌开发的开源移动操作系统 | Android 11, Android 10, Android 9 Pie |
| **iOS** | 苹果开发的专有移动操作系统 | iOS 14, iOS 13, iOS 12 |

## 2. Linux系统目录结构

Linux采用树形目录结构，所有文件和目录都从根目录（/）开始。

### 重要目录说明

| 目录 | 描述 |
|------|------|
| **/boot** | 存放启动Linux时使用的内核文件、连接文件及镜像文件 |
| **/etc** | 存放系统配置文件和子目录列表 |
| **/lib** | 存放基本代码库，类似Windows的DLL文件 |
| **/sys** | 集成proc、devfs和devpts文件系统信息，反映内核设备树 |
| **/bin** | 存放最常用的程序和指令（如ls） |
| **/sbin** | 存放只有系统管理员能使用的程序和指令 |
| **/dev** | 存放Linux的外部设备 |
| **/media** | 存放识别后的其他设备（如U盘、光驱） |
| **/mnt** | 临时挂载其他文件系统的目录 |
| **/run** | 临时文件系统，存储系统启动以来的信息 |
| **/lost+found** | 系统非法关机后存放文件的目录 |
| **/tmp** | 存放临时文件的目录 |
| **/root** | 系统管理员的用户主目录 |
| **/home (~)** | 用户的主目录，以用户账号命名 |
| **/usr** | 存放用户应用程序和文件，类似Windows的Program Files |
| **/usr/bin** | 存放系统用户使用的应用程序与指令 |
| **/usr/sbin** | 存放超级用户使用的高级管理程序和系统守护程序 |
| **/usr/src** | 内核源代码的默认放置目录 |
| **/var** | 存放经常修改的数据，如程序运行的日志文件 |
| **/proc** | 虚拟目录，管理内存空间，是系统内存的映射 |
| **/opt** | 存放额外安装的软件 |
| **/srv** | 存放服务启动后需要提取的数据 |

### 系统启动流程
1. **BIOS开机自检**
2. 按照BIOS设置的启动设备启动
3. 操作系统接管硬件，读入 **/boot 目录下的内核文件**
4. 运行init进程（系统所有进程的起点）
5. **sync命令**：将内存中的数据与磁盘同步，确保数据一致性

## 3. Linux命令基础

### 命令格式
```bash
command [-options] [parameter1] [parameter2] ...
```

| 组成部分 | 描述 | 示例 |
|----------|------|------|
| **命令** | 要执行的命令或操作 | `ls` |
| **选项** | 指定命令的具体行为或修改默认行为 | `-l` |
| **参数** | 提供命令操作所需的内容或数据 | `/home` |

### 常见Linux命令

| 命令 | 英文全拼 | 描述 |
|------|----------|------|
| `ls` | list files | 列出目录内容，包括文件和子目录 |
| `cd` | change directory | 切换当前工作目录 |
| `pwd` | print work directory | 显示当前工作目录的完整路径 |
| `mkdir` | make directory | 创建一个新的目录 |
| `rmdir` | remove directory | 删除一个空的目录 |
| `cp` | copy file | 复制文件或目录到指定位置 |
| `rm` | remove | 删除文件或目录 |
| `mv` | move file | 移动文件或目录到指定位置，或重命名 |

### 常用命令选项

| 选项 | 描述 |
|------|------|
| `-a` | 列出所有文件，包括隐藏文件（以`.`开头） |
| `-d` | 仅列出目录本身，而非目录内的文件内容 |
| `-l` | 长格式显示，包括文件属性、权限、大小等信息 |

## 4. Linux文件权限管理

### 文件属性解读
Linux文件权限由10个字符组成：
- 第1个字符：文件类型（`d`表示目录，`-`表示文件）
- 第2-4个字符：所有者权限
- 第5-7个字符：用户组权限
- 第8-10个字符：其他用户权限

### 权限字符含义
- `r`（read）：读权限，数值为4
- `w`（write）：写权限，数值为2
- `x`（execute）：执行权限，数值为1
- `-`：无权限，数值为0

### 权限计算示例
权限 `-rwxrwx---` 的数值计算：
- owner = rwx = 4+2+1 = 7
- group = rwx = 4+2+1 = 7
- others = --- = 0+0+0 = 0

### 权限修改命令

#### chown（change owner）
修改文件所属用户与组：
```bash
chown [-R] 所有者:属组名 文件名
# 示例
chown bin install.log
```

#### chmod（change mode）
修改用户权限：
```bash
chmod [-R] xyz 文件或目录
# 示例：设置权限为755
chmod 755 filename
```

## 5. Linux用户和用户组管理

### 用户管理命令

| 命令 | 描述 | 主要选项 |
|------|------|----------|
| **useradd** | 添加新用户账号 | `-c` 注释<br/>`-d` 指定主目录<br/>`-g` 指定用户组<br/>`-G` 指定附加组<br/>`-s` 指定登录Shell<br/>`-u` 指定用户号 |
| **userdel** | 删除用户账号 | `-r` 同时删除用户主目录 |
| **usermod** | 修改用户信息 | `-c` `-d` `-m` `-g` `-G` `-s` `-u` `-o` `-l` 新用户名 |
| **passwd** | 修改用户口令 | `-l` 锁定口令（禁用账号）<br/>`-u` 口令解锁<br/>`-d` 使账号无口令<br/>`-f` 强制下次登录修改口令 |

### 用户组管理
- 用户组的所有信息存放在 `/etc/group` 文件中
- 通过用户组可以方便地管理多个用户的权限

## 6. Linux磁盘管理

### 磁盘管理三大命令

| 命令 | 英文全称 | 描述 |
|------|----------|------|
| **df** | disk free | 列出文件系统的整体磁盘使用量 |
| **du** | disk used | 检查磁盘空间使用量 |
| **fdisk** | - | 用于磁盘分区 |

### 磁盘操作命令

#### 磁盘格式化
```bash
mkfs [-t 文件系统格式] 装置文件名
```

#### 磁盘检验
```bash
fsck [-t 文件系统] [-ACay] 装置名称
```
- **fsck**（file system check）：检查和维护不一致的文件系统
- 用于系统掉电或磁盘问题后的文件系统检查

## 8. 重要概念和技巧

### 目录表示法
- `.`：代表当前目录，也可用 `./` 表示
- `..`：代表上一层目录，也可用 `../` 表示
- 以点（`.`）开始的文件或目录是隐藏文件（如 `.bashrc`）

### 服务器环境
常见的Web服务器组合：
- **LAMP**：Linux + Apache + MySQL + PHP
- **LNMP**：Linux + Nginx + MySQL + PHP

### 文件和目录特点
- 隐藏文件：以点（`.`）开头的文件或目录
- 默认查找时不显示隐藏文件
- 用户组信息存储在 `/etc/group` 文件中

## 9. 最佳实践

### 系统管理
1. **定期备份重要数据**
2. **合理设置文件权限**，遵循最小权限原则
3. **定期检查磁盘空间**使用情况
4. **及时更新系统**和软件包

### 安全建议
1. **不要以root用户**进行日常操作
2. **设置强密码**并定期更换
3. **合理配置防火墙**规则
4. **监控系统日志**，及时发现异常

### 命令使用技巧
1. **使用Tab键**自动补全命令和路径
2. **使用历史命令**（上下箭头键）提高效率
3. **善用管道和重定向**处理数据
4. **学习正则表达式**进行高效文本处理
